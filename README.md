# 📧 Email Spam Detection

This project is a machine learning model that detects whether an email is spam or not.

## 🚀 Features
- Data preprocessing
- Text cleaning
- Model training
- Spam classification

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## ▶️ How to Run
1. Open the notebook file
2. Run all cells step by step
3. Check the output

## 📊 Dataset
- Email dataset used for spam detection
